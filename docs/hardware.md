# 硬件设计

## 1. 形态：智能台灯

- 不是「一台设备」，而是「一盏本来就该在书桌上的台灯」——小朋友接受度高。
- 灯臂 = 天然云台，提供俯视作业的视角。
- 光源可控 = 视觉的最大福利：可控光照下的识别比「野生摄像头」稳一个量级。

## 2. 双摄分工

```
        [灯头摄像头] ← 俯视作业，略偏 10~20°（防灯光镜面反射）
         \
          \
   ┌──────┐\
   │灯臂   │ \
   │      │  ▼  📄 作业纸
 ┌─┴─┐
 │柱 │[坐姿摄像头] → 看上半身：头-桌距离 / 肩膀歪斜 / 驼背
 │子 │  约 30~40cm 高，3/4 正侧角度略俯
 │Pi │
 └───┘  ==== 书桌面 ====
```

| 摄像头 | 位置 | 看什么 | 摆位要点 |
|--------|------|--------|----------|
| 坐姿 | 柱子（和 Pi 一起） | 上半身姿态 | 3/4 正侧视、30~40cm 高、略俯 |
| 作业 | 灯头 | 作业 OCR + 顺带握笔 | 别正俯拍，偏 10~20° 让反光弹开镜头 |

握笔检测复用**灯头那个俯视摄像头**（手和笔本就在它视野里），不加第三个摄像头。

## 3. BOM（建议）

| 部件 | 原型（现有） | 量产形态 | 备注 |
|------|--------------|----------|------|
| 主板 | **Pi 3**（手上有） | **Pi 5 8GB** | 8GB 是地板；别为省钱降 4GB |
| 摄像头 | 1× USB 1080p | 柱子 CSI + 灯头 USB | Pi 3 只有 1 个 CSI；USB 更省心 |
| 屏幕 | HDMI 小屏 / 复用 | 7" 官方触摸屏 (800×480) | 当低速反馈面板，**不是视频监视器** |
| 音频 | USB 麦 + 喇叭 | ReSpeaker 阵列麦（Pi 5） | DSP 进阶可换降噪算法 |
| 电源 | Pi 3 原电源 | **Pi 5 需 5V/5A (27W) USB-C** | Pi 3 电源带不动 Pi 5 |
| 散热 | 散热片 | 主动散热风扇 | Pi 与 LED 的热要分开 |
| 灯臂 | 现成鹅颈夹子灯 | 带锁定关节的节臂 | 鹅颈晃 → 作业摄像头运动模糊 |
| 外壳 | 纸盒 / 现成灯改造 | Tinkercad 设计 + 代打印 | **别开模**（量产才开模，几万块） |

## 4. 几个会踩的坑

1. **别开模**：开模 = 注塑钢模、量产上千件才划算。你是单件原型，用 3D 打印/拼装。
   不会画图也能做：改造现成鹅颈灯（零画图）、下载现成模型、或 Tinkercad（和娃拖积木）。
2. **CSI 线长**：Pi 标准排线 15~50cm，灯臂够不着 → 灯头用 USB（可拉 3~5m）。
3. **灯头反光**：俯拍作业别正垂直，偏 10~20°，避免灯光镜面反射糊成一片（=避免高光削顶）。
4. **散热**：Pi + LED 驱动都发热，柱子底座要开散热孔，热源分开。
5. **屏幕定位**：屏幕显示提示/对话/偶发抓拍，**不要实时镜像摄像头**（既无用又费内存带宽）。

## 5. 采购纪律

- Pi 5 买**裸板 8GB**，别买 3000 的全家桶套餐；配件优先复用 Pi 3 的。
- 钱花在**摄像头画质 + 灯臂刚性 + 光照**上——这些才真正决定识别效果。
